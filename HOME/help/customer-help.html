Customer help

1. Browse & filters

Open Marketplace. Use search, price range, category, farmer name, and location filters. Pagination is at the bottom.

2. Add to cart & call farmer

Each product card shows the farmer phone (when the farmer added it at signup). Use Add to cart to build your list, then open Cart from the marketplace header. You can still call the farmer directly to confirm weight, delivery, and payment.

3. Dashboard & cart

After login, use Customer dashboard for another product grid and cart count. View Cart shows line items and farmer numbers at checkout.

4. Run the demo

Start MongoDB, then cd server → npm start → visit http://localhost:5000/ so cookies and images work on one port.
