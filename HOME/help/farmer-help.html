किसान सहायता (Farmer help)

1. अकाउंट

साइनअप पर 10 अंकों का मोबाइल (6–9 से शुरू) और ईमेल verify के बाद ही OTP मांगें। लॉगिन के बाद Farmer dashboard खुलेगा।

2. उत्पाद जोड़ना

Add Product से नाम, दाम, मात्रा भरें। फोटो अपलोड वैकल्पिक है। सेव के बाद फॉर्म खाली हो जाएगा — आप कई उत्पाद लगातार जोड़ सकते हैं। View Inventory से सूची देखें।

3. ग्राहक आपको कैसे कॉल करेगा

आपके प्रोफाइल में जो फोन नंबर साइनअप पर दिया है, वह मार्केटप्लेस पर उत्पाद के साथ दिखेगा ताकि ग्राहक सीधे ऑर्डर के लिए कॉल कर सके। नंबर अपडेट करने के लिए नया अकाउंट या भविष्य में प्रोफाइल API जोड़ा जा सकता है।

4. सर्वर चलाना

MongoDB चालू करें, फिर टर्मिनल में: cd server → npm start → ब्राउज़र में http://localhost:5000/
